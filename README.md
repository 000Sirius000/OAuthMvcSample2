# OAuthMvcSample  
**Кросплатформене програмування — Лабораторна №3–4**  
**Автор:** Тарнавський Микита Олегович, група ІПЗ-34  

Цей вебзастосунок реалізує автентифікацію та авторизацію користувачів з використанням ASP.NET Core MVC, Identity, OpenIddict (OAuth2/OpenID Connect), а також демонструє побудову ORM рівня даних за підходом **Code First** із можливістю вибору типу СУБД та виконанням складних запитів із фільтрацією.

---

## Функціональні можливості застосунку

### 1. Авторизаційна частина (Лабораторна №3)
- Реєстрація користувача  
- Вхід за логіном та паролем  
- Перегляд профілю користувача  
- Зміна даних та пароля  
- Захист внутрішніх сторінок через `[Authorize]`
- Використання **OpenIddict** як **Authorization Server** та **Resource Server**

---

### 2. Доменна частина (Лабораторна №4)

Реалізована ORM на основі **Entity Framework Core (Code First)** з можливістю конфігураційно змінювати тип БД на один із:

| Тип БД | Статус |
|--------|---------|
| MS SQL Server | ✔ |
| PostgreSQL | ✔ |
| SQLite | ✔ |
| In-Memory | ✔ |

Тип БД задається у файлі `appsettings.json`:

```json
"Database": {
  "Provider": "Sqlite" // або SqlServer, Postgres, InMemory
}
